1. 콘텐츠 개요
콘텐츠명: 파이썬 학습 프로그램
대상: 프로그래밍 입문자
목표: 기초 문법부터 함수까지 학습
학습 방식: 개념설명 → 예제 → 실습 → 퀴즈 → 다음 챕터

2. 챕터 구성
챕터0: 출력 (print)
챕터1: 변수 (int, float 등)
챕터2: 입력 (input)
챕터3: 연산자 (+,- 등)
챕터4: 조건문 (if)
챕터5: 반복문 (while)
챕터6: 함수 (def)

3. 템플릿
- 개념설명: 학습 목표, 학습 내용, 설명
- 따라 써보기: 예제
- 실습 문제: 문제 상황, 스켈레톤 코드, 힌트
