1. 시스템 개요
이 프로젝트는 사용자 중심의 단계별 파이썬 기초 학습 프로젝트로 메인 컨트롤러가 중앙 제어를 담당하고 각 학습 챕터를 모듈화하여 독립적으로 로드하는 중앙 집중형 모듈러 아키텍처를 채택한다. 새로운 학습 단원이 추가되더라도 메인 로직의 수정을 최소화하도록 설계한다.

2. 디렉토리 구조 명세 (*변경됨*)
learn-python/
│
├── main.py               # [Controller] 프로그램 진입점 및 챕터 라우팅 관리
└── chapters/             # [Model/View] 독립적 학습 콘텐츠 모듈 폴더
    ├── __init__.py       # 폴더를 파이썬 패키지로 정의하는 초기화 파일
    ├── chapter0.py       # 출력 (print) 콘텐츠 엔진
    ├── chapter1.py       # 변수 (Variable) 콘텐츠 엔진
    ├── chapter2.py       # 입력 (input) 콘텐츠 엔진
    ├── chapter3.py       # 연산자 (Operator) 콘텐츠 엔진
    ├── chapter4.py       # 조건문 (Conditional) 콘텐츠 엔진
    ├── chapter5.py       # 반복문 (Loop) 콘텐츠 엔진
    └── chapter6.py       # 함수 (Function) 콘텐츠 엔진
