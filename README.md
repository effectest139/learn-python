# 프로젝트 개요
- 목표: Python을 배우는 학생들을 위한 대화형 학습 플랫폼
- 진로 연계: 컴퓨터 교육과 지망으로, 향후 교육용 소프트웨어 개발 경험

# 기술 스택
- 언어: Python
- GUI: Tkinter
- 버전관리: Git/GitHub
- 개발 도구: VS Code

# 프로젝트 구조
- chapters/: 학습 콘텐츠 (7개 챕터)
- engine/: 실행 엔진 (콘솔 + GUI 동시 지원)
- main.py / app.py: 진입점

# 사용된 Python 문법
- 변수, 함수, 조건문, 반복문, 딕셔너리/리스트
- 파일 I/O (exp_manager.py)
- 클래스와 메서드 (ChapterFrame)
- 모듈 import와 패키지 구조

# 실행 방법
python main.py (콘솔 버전)
python app.py (GUI 버전)
